#### ZTBCMS v4基于TP6开发，文档请移步到 https://www.kancloud.cn/ztbcms/ztbcms/1939308

### 特点

* 基于 [ThinkPHP 3.2](https://www.kancloud.cn/manual/thinkphp) 框架开发，采用独立分组的方式开发的内容管理系统
* 支持模块安装/卸载，拥有丰富的[扩展模块](http://mod.ztbcms.com/)
* 同时系统对扩展方面也支持比较大，可以使用内置的行为控制，对现有功能进行扩展
